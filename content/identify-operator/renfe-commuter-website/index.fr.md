---

Sur le site Web Cercanías de Renfe (trains régionaux), il est possible de rechercher des correspondances dans chaque réseau de trains régionaux. Il faut sélectionner la région correspondante sur le site, puis cliquer sur _Horaires_. Dans les réseaux régionaux avec plusieurs opérateurs (par exemple, Barcelone), seuls les trains de Renfe sont affichés.
