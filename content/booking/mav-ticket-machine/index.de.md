Fahrkartenautomaten der MÁV gibt es an vielen größeren Bahnhöfen. Hier werden FIP 50 Tickets und Reservierungen für die MÁV und GySEV verkauft.

{{% booking-section "fip_50" %}}

## FIP 50 Fahrkarten

Am MÁV Ticketschalter können FIP 50 Fahrkarten für Reisen innerhalb Ungarns für die MÁV sowie GySEV und die grenzüberschreitende Verbindungen erworben werden.

Zum Buchen von FIP 50 Tickets muss bei den Passergiervergünstigungen die Option _Further Discounts_ gewählt werden. Anschließend kann _FIP 50% domestic discount 1st class (50%)_ oder _FIP 50% domestic discount 2nd class (50%)_ gewählt werden. Die Klasse bezieht sich hierbei noch nicht auf die gewünschte Klasse des Tickets, sondern der Berechtigung.

## Reservierungen

Reservierungen für Züge der MÁV sowie GySEV können vor Ort zu einem Preis von 990 HUF erworben werden.

Zum Buchen von Reservierungen ohne Ticket muss in der Übersicht der Reisenden _Further settings_ gewählt werden. Hier kann anschließend die Option _I only need a seat reservation_ ausgewählt werden.
