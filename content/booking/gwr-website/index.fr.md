## Réservations

Sur le site web de GWR, il est possible d’effectuer des réservations de places gratuites pour tous les services ferroviaires nationaux où des réservations sont possibles.

Pour réserver une place, il faut créer un compte utilisateur. Après une connexion réussie, une liaison peut être sélectionnée et réservée dans le menu, dans la section _My journeys_, via le bouton _Make a reservation_.
