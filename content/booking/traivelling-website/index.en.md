## FIP Global Fare

Traivelling sells FIP Tickets for Eurostar Blue and Eurostar Red on their website.

{{% float-image
    src="step_1.webp"
    caption="Step 1: Enter connection details."
    width="50%"
    position="right"
%}}

In step 1, first select the connection and enter the number of travellers. The discount card must be entered individually for each traveller.

{{% /float-image %}}

{{% float-image
    src="step_2.webp"
    caption="Step 2: Enter discount card."
    width="50%"
    position="right"
%}}

In step 2, search for "FIP" in the search field and find the option "FIP International Reduced Rate Card" (if the website is in English) or "FIP 50%" (for all other languages) in the "Austria" section.

{{% /float-image %}}

{{% float-image
    src="step_3.webp"
    caption="Step 3: Confirm connection."
    width="50%"
    position="right"
%}}

In step 3, search for connections. The discount card is automatically taken into account and the price is adjusted accordingly.

{{% /float-image %}}

{{% float-image
    src="step_4.webp"
    caption="Step 4: Check fare."
    width="50%"
    position="right"
%}}

After selecting the connection, make sure that "FIP Leisure" is listed as the fare. If this is not the case, it is a public fare with the corresponding cancellation conditions.

{{% /float-image %}}

{{% highlight "important" %}}
If the journey starts or ends in Germany, no booking fee is charged. In all other cases, a booking fee of 8 % is charged.
{{% /highlight %}}
