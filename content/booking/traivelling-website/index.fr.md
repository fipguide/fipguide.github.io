## Tarif Global FIP

Traivelling vend des Billets FIP pour Eurostar Blue et Eurostar Red sur son site Web.

{{% float-image
    src="step_1.webp"
    caption="Étape 1 : Saisir les données de connexion."
    width="50%"
    position="right"
%}}

À l'étape 1, sélectionnez d'abord la connexion et indiquez le nombre de voyageurs. La carte de réduction doit être renseignée individuellement pour chaque voyageur.

{{% /float-image %}}

{{% float-image
    src="step_2.webp"
    caption="Étape 2 : Renseigner la carte de réduction."
    width="50%"
    position="right"
%}}

À l'étape 2, recherchez « FIP » dans le champ de recherche et trouvez dans la section « Autriche » l'option « FIP International Reduced Rate Card » (si le site est en anglais) ou « FIP 50% » (pour toutes les autres langues).

{{% /float-image %}}

{{% float-image
    src="step_3.webp"
    caption="Étape 3 : Confirmer la connexion."
    width="50%"
    position="right"
%}}

À l'étape 3, recherchez les connexions. La carte de réduction est automatiquement prise en compte et le prix est ajusté en conséquence.

{{% /float-image %}}

{{% float-image
    src="step_4.webp"
    caption="Étape 4 : Vérifier le tarif."
    width="50%"
    position="right"
%}}

Après avoir sélectionné la connexion, vérifiez impérativement que le tarif indiqué est « FIP Leisure ». Si ce n'est pas le cas, il s'agit d'un tarif public avec les conditions d'annulation correspondantes.

{{% /float-image %}}

{{% highlight "important" %}}
Si le trajet commence ou se termine en Allemagne, aucun frais de réservation n'est appliqué. Dans tous les autres cas, des frais de réservation de 8 % sont facturés.
{{% /highlight %}}
