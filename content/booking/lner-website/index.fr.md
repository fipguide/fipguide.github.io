## Réservations

Les réservations ne peuvent être effectuées que sur le site britannique de LNER (www.lner.co.uk). Lors de la visite du site, il faut impérativement sélectionner l'option « Stay on lner.co.uk » dans la fenêtre pop-up.

![Site Web LNER rester sur lner.co.uk](lner-website.webp)

Sur le site LNER, des réservations de sièges gratuites peuvent être effectuées pour les liaisons LNER. Pour cela, sélectionnez _Reserve Spaces_ sur la page d'accueil et saisissez les détails du voyage. Après la sélection d'un train, le site demande une confirmation de réservation. Sélectionnez _My booking is not listed_ puis _Other_. Le formulaire demande également une référence de réservation, mais n'importe quelle valeur peut être saisie. Nous recommandons d'indiquer « FIP Coupon » sous _additional details_.

Après avoir sélectionné _Reserve a seat_ et/ou _Reserve a bike space_, la réservation est émise sous forme de billet électronique qui peut être affiché dans l'application et ajouté au portefeuille mobile. Un siège est automatiquement attribué, mais peut être modifié par la suite.
