## Reservierungen

Reservierungen können nur auf der britischen Seite von LNER (www.lner.co.uk)vorgenommen werden. Beim Besuch der Seite muss daher auf jeden Fall die Option "Auf lner.co.uk bleiben / Stay on lner.co.uk" im Pop-up-Fenster ausgewählt werden.

![LNER Website auf lner.co.uk bleiben](lner-website.webp)

Auf der LNER Website können kostenlose Sitzplatzreservierungen für LNER-Verbindungen vorgenommen werden. Dazu auf der Startseite _Reserve Spaces_ auswählen und die Reiseverbindung eingeben. Nach Auswahl eines Zuges fragt die Website nach einer Buchungsbestätigung. _My booking is not listed_ und dann _Other_ auswählen. Das Formular erfordert außerdem eine Buchungsreferenz, es kann jedoch ein beliebiger Wert eingegeben werden. Wir empfehlen bei _additional details_ "FIP Coupon" anzugeben.

Nach Auswahl von _Reserve a seat_ und/oder _Reserve a bike space_ wird die Reservierung als E-Ticket ausgestellt, das in der App angezeigt und zum mobilen Wallet hinzugefügt werden kann. Ein Sitzplatz wird automatisch zugewiesen, kann aber nachträglich geändert werden.
