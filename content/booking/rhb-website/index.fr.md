## Billets FIP 50

Sur le site web de RhB, des Billets FIP 50 peuvent être achetés pour la liaison Tirano – St. Moritz (ligne de la Bernina) avec la réduction « Half Fare Travelcard ».[^1] Cette option est en principe destinée aux voyageurs disposant d’un abonnement Half Fare (carte de réduction des CFF), mais elle est aussi acceptée comme solution simple pour les Billets FIP 50. Le tarif Half Fare doit être sélectionné lors de la saisie des voyageurs dans le processus de réservation.

## Réservations

Les réservations pour la liaison Tirano – St. Moritz (ligne de la Bernina) peuvent être achetées sur le site web de RhB. Dans le processus de réservation, l’option « Add seat reservation » doit être sélectionnée. La réservation coûte 5 CHF par personne.

Si seule une réservation sans billet est souhaitée, l’option « This passenger only needs a seat reservation. » peut être sélectionnée lors de la saisie des voyageurs.
